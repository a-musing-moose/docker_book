.. _glossary:

Glossary
========


.. glossary::
    :sorted:

    Virtual Machine
        A virtualisation technology mostly focused on virtualising the hardware via a
        :abbr:`HAL (Hardware Abstraction Layer)`

    OS
        Operating System

    Container
        A process running in a light weight isolation system

    PaaS
        Platform as a Service.

    FaaS
        Function as a Service.

    Swarm Mode
        A native multi-server clustering system built into Docker

    Docker Engine
        The actual runtime for Docker Containers

    Docker Machine
        A tool for installing and managing remote Docker hosts or provising a swarm cluster

    Docker
        The company behind the Docker Engine, and other tooling. Informally used to refer to the Docker Engine itself

    Image
        The blueprints for a container
